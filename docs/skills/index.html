BECOMING A PUBLIC ENTREPRENEUR

INNOVATION SKILLS ACCELERATOR



Welcome to the New Jersey State Innovation Skills Accelerator. The Innovation Skills Accelerator is a free, online program to train public entrepreneurs to use innovative methods, including design thinking, evidence-based decisionmaking and collective intelligence, to solve public problems. Each module includes video instruction, interviews with leading public sector innovators from around the world, readings and self-assessments.
